# Newsletter Production Agent Guide

Route work by stage:

- Use `01_intake/` for audience, source notes, and issue goals.
- Use `02_draft/` for turning approved notes into newsletter copy.
- Use `03_review/` for final checks before handoff.

Read the root `CONTEXT.md` first, then read the local stage `CONTEXT.md` before working in that stage.
Do not invent quotes, metrics, or source facts.
